## üöÄ Getting Started

To get a local copy up and running, follow these simple steps:

1. Clone the repository
   ```
   git clone https://github.com/yourusername/jsonit.git
   ```
2. Install NPM packages
   ```
   npm install
   ```
3. Run the development server
   ```
   npm run dev
   ```
